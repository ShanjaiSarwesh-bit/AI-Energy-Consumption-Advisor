# AI-Based Smart Energy Consumption Advisor

## Problem Statement
Colleges consume large amounts of electricity without proper analysis, leading to energy wastage and higher costs.

## Solution
An AI-based advisory system using prompt engineering, RAG, and agentic AI workflows to analyze energy data and provide sustainability-focused recommendations.

## Technologies Used
- Prompt Engineering
- Retrieval-Augmented Generation (RAG)
- Agentic AI
- IBM Granite Models (conceptual)
- Conversational AI

## SDG Alignment
- SDG 7: Affordable and Clean Energy
- SDG 13: Climate Action
